FIZYON DESIGN SYSTEM · v1 (proposed)

Calm, clinical, trustworthy.

Token-driven foundation. Web-first, scales to mobile. Reserve red for true errors; secondary text passes WCAG AA; every interactive element has a designed focus ring. Display face (Fraunces) used with restraint for headings and key numbers only.

Color

Healing teal brand + coral accent (sparingly). Semantic colors are calm; amber = “needs attention”, red = true error only.
